# Simple Tic-Tac-Toe game #

## Game rules ## 
  ### - Players take turn putting their marks in empty squares ###

  ### - Winning conditions ###
``` 
[
  // Rows
  [0, 1, 2],
  [3, 4, 5],
  [6, 7, 8],
  // Columns
  [0, 3, 6],
  [1, 4, 7],
  [2, 5, 8],
  // Diagonals
  [0, 4, 8],
  [2, 4, 6]
];
```
